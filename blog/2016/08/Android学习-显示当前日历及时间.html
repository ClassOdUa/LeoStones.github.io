Android学习-显示当前日历及时间

发表于 2016-08-17   |   分类于 Android学习之路   |     |   本文总阅读量 次

学习目标

  • DatePicker && TimePicker 动态输入日期和时间的功能
  • DatePickerDialog && TimePickerDialog 动态输入日期和时间的对话框
  • 监测日期时间改变的监听器：
    1>OnDateChangedListener && OnTimeChangedListener()
    2>OnDateSetListener && OnTimeSetListener()

对应事件

DatePicker——OnDateChangedListener—–OnDateChange() 事件
TimePicker——OnTimeChangedListener—–OnTimeChange() 事件
DatePickerDialog——OnDateSetListener—–OnDateSet() 事件
TimePickerDialog——OnTimeSetListener—–OnTimeSet() 事件

注意事项

  • DatePicker 对象是以 init 这个方法初始化来指定 DatePicker 初始化的年月日以及 OnDateChangedListener 来监听日期的改变，而 TimePicker 不需要 init 初始化，只需要设置监听 OnTimeChangedListener 来监听时间的改变

    ​

  • DatePickerDailog 是以对话框的方式弹出日期选择器，需要用 new 的方式创建并且传入对应的监听 OnDateSetListener，日期初始化数据，最后记得设置对话框显示 (show) 方法。同样 TimePickerDailog 也是需要以 new 的方式进行初始化，并且设置 OnTimeListener,
